## Savol 1



PIP nima?



- [A] Python dasturlash tilining o'zi  

- [B] Python uchun paket menejeri  

- [C] Python kodi tahrirlash dasturi  

- [D] Python virtual muhiti  



> **To'g'ri javob:** B  

> **Tushuntirish:** PIP (Pip Installs Packages) — bu Python uchun rasmiy paket menejeri bo'lib, PyPI (Python Package Index) dan paketlarni o'rnatish, yangilash va o'chirishga imkon beradi.

## Savol 15



`pip check` buyrug'i nima uchun ishlatiladi?



- [A] PIP yangi versiyasini tekshirish uchun  

- [B] O'rnatilgan paketlardagi bog'liqlik muammolarini aniqlash uchun  

- [C] Internet ulanishini tekshirish uchun  

- [D] Python versiyasini tekshirish uchun  



> **To'g'ri javob:** B  

> **Tushuntirish:** `pip check` buyrug'i o'rnatilgan paketlar orasidagi bog'liqlik (dependency) muammolarini aniqlaydi — ya'ni biror paket o'z bog'liqliklarini to'liq qondirayotgan yoki yo'qligini tekshiradi.
